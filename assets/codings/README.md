# Add coding project thumbnails here

Put thumbnail image files (`.jpg`, `.png`, `.webp`) in this folder, then reference them
from `data/codings.json` using the path `assets/codings/your-file.jpg`.

For playable projects, set `"type": "embed"` and add an `"embedUrl"` (for example a
Scratch embed link). If an image is missing, a friendly placeholder is shown.
